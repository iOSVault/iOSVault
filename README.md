iOSVault is a web platform for exploring and downloading classic iOS applications and games that can be run using the TouchHLE emulator. This project focuses on preserving legacy iOS apps so they remain accessible on modern devices.

---

How It Works

iOSVault fetches data directly from Internet Archive using its metadata API, then processes and displays a list of available applications that users can browse and download.

---

Data Source

All data is sourced from:

- Internet Archive (archive.org)

iOSVault does not host any files on its own servers.
